# Office20 Free Sample

## 職場英語フレーズカード 10

日米の職場でよく使う英語表現を、短く・自然に・すぐ使える形でまとめた無料サンプルです。

対象:

- アメリカの職場で働く予定の方
- 英語メールや会議で自然な表現を増やしたい方
- 日本語の直訳ではなく、実務で使いやすい言い方を知りたい方

使い方:

- 表面: 日本語の場面を見て、英語でどう言うか考える
- 裏面: 英語表現、ニュアンス、例文を確認する
- CSV版はAnki、Quizlet、Google Sheetsなどに取り込めます

有料版の展開案:

- 職場英語フレーズカード100
- メール表現編
- 会議表現編
- 上司・同僚との会話編
- HR・学校・手続き編
